# Write a function to map two lists into a dictionary

Define a function that takes two lists as parameter and returns a dictionary after combining the lists.

_Example:_


**Input:**


`[1, 2, 3], [10, 20, 30]`


**Output:**


`{1: 10, 2: 20, 3: 30}`



 _Challenge:_
 
Write the function using only **one** for-loop.
